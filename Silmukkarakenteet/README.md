## Silmukkarakenteet
- Silmukkalauseet ovat yksinkertainen mutta tehokas keino toistaa ohjeita niin kauan kuin tietty ehto (lause) on täytetty.
- do-while
- while
- for
- foreach(Foreach-lause toistaa lauseen tai lohkon lauseita jokaiselle elementille taulukossa tai kokoelmassa.)
- Goto-lause siirtää ohjelman suorituksen tietyn nimikkeen (label) kohdalle. Sitä voidaan käyttää suorituksen siirtämiseksi toiseen switch-case -nimikkeeseen, tai mihin tahansa muualle kohtaan lähdekoodia, vaikkapa sisäkkäisten silmukoiden tai muiden vastaavanlaisten rakenteiden välttämiseksi.
- Continue-lause ohjaa suorituksen jatkumaan suoraan silmukan seuraavasta kierroksesta (mikäli ehto evaluoituu yhä todeksi), jättäen suorittamatta välissä olleen koodin (toisin sanoen lopun käynnissä olleesta kierroksesta).
- Break
- Return
